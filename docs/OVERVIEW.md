# Nordic Global Solutions — Overview

## Mission

We are seeking people to build the future of technology. Nordic Global Solutions develops core platforms that help people and teams work smarter.

## Three Core Platforms

| Platform | Description |
|----------|-------------|
| **Ona** | Operating system for artificial intelligence. Your AI teammate. |
| **WorkFlowSpace** | Unified workspace for teams. One place for calendar, chat, tasks, docs. |
| **Affiliate Flow** | Marketing automation platform. Landing pages, social content, email/SMS outreach. |
| **BitNexus** | Decentralized affiliate revenue platform. AI-powered, MEV/XAB staking, NXC token, MLM network. Launch Jan 21, 2026. |
