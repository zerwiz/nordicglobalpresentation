## AI Dev Suite

100% local AI tooling for developers. Knows your files and code; helps you develop with AI.

### Features

- **One-click install**: Zed, OpenCode, Ollama, LM Studio
- **Chat with Ollama** — extended thinking (Qwen3, DeepSeek R1/V3) for reasoning traces
- **Zed & OpenCode via ACP** — Agent Client Protocol, local inference
- **Memory, document drive, RAG research** — hybrid BM25+vector retrieval
- **Chat commands**: `/memory`, `/remember`, `/behavior`, `/drive add`, `/research`, `/bye`
- **TUI, web, desktop app** (Electron)

### Components

| Component | Description |
|-----------|-------------|
| **Electron app** | Standalone UI (React + WhyNot styling). Chat, Drive, Memory, Tools, Settings. Spawns Elixir API. Works offline with Ollama. |
| **Debugger & Observer** | Diagnose "(no response)", "terminated", "Cannot reach API". Status, logs, test chat, LLM analysis. Terminal observer. A2A agent. |
| **RAG system** | Index PDF, DOCX, TXT, MD. Hybrid BM25+vector. Query with citations. Web research. Chroma + Ollama. |
| **Memory & Drive** | Persistent context. `/remember`, auto-extracted facts, `/drive add`. Stored in `~/.config/ai-dev-suite/`. |

### HTTP API

REST at `http://localhost:41434`. Chat, stream, memory, behavior, knowledge bases, research. CORS enabled.

Key endpoints: `/api/chat`, `/api/chat/stream`, `/api/ollama/models`, `/api/knowledge-bases`, `/api/memory`, `/api/research`.
